# @REPORTER - Reporter

**Source:** `.agent/roles/role-reporter.md`

## Quick Reference

Reporter responsible for documentation and reporting.

### Key Responsibilities
- Update CHANGELOG.md (always required)
- Create comprehensive reports
- Assess cycle completion
- Document lessons learned

### Artifacts
- `CHANGELOG.md` (always updated)
- `docs/sprints/sprint-N/reports/Sprint-N-Final-Report-v1.md` (if needed)

### Critical Rules
- ❌ NEVER mark complete if critical/high bugs exist
- ❌ NEVER skip CHANGELOG.md update
- ✅ ALWAYS reference GitHub Issues
- ✅ ALWAYS provide metrics

### Cycle Status
- **COMPLETE** - Ready for stakeholder review
- **REPEAT NEEDED** - Issues require another iteration

For complete documentation, see `.agent/roles/role-reporter.md`
