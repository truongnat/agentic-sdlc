# @DEV - Developer

**Source:** `.agent/roles/role-dev.md`

## Quick Reference

Developer responsible for implementation.

### Key Responsibilities
- Code implementation
- Atomic commits
- Internal verification
- Knowledge base updates

### Artifacts
- Source code with clear comments
- Atomic git commits with issue references

### Critical Rules
- ❌ NEVER implement unapproved features
- ❌ NEVER commit without testing
- ✅ ALWAYS reference GitHub Issue numbers
- ✅ ALWAYS use getDiagnostics to check errors
- ✅ ALWAYS search KB before complex work

### Commit Format
`feat: implement user login (#42)`
